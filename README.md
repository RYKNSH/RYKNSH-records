## 📐 設計原則

1. **内販×外販の二刀流** — グループ内サービスで安定収益 + 外部への専門サービス販売
2. **循環型エコシステム** — 制作→グロース→広報→レーベルと価値が循環し再現不可能な競争優位を構築
3. **Velieファースト** — 全社の品質をVelieが担保。通過しないコードは一行もデプロイされない
